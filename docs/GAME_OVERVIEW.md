# Game Overview

`AITown` is organized as a lightweight management and simulation game prototype.

## Design Goals

- keep the codebase compact and readable
- support a complete playable loop
- combine simulation systems with accessible controls
- leave room for packaging and release automation

## Core Gameplay Layers

- player movement and interaction
- chapter-based progression
- NPC-driven social and economic activity
- zone upgrades and production/trade loops
- world-state feedback through UI and logs

## Engineering Goals

- deterministic enough for smoke testing
- simple asset pipeline
- clear folder layout
- explicit release validation before packaging
