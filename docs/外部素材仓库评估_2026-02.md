## 结论（用于 Steam 商业发布）

- `StardewValley-Assets`：**禁止直接使用**。仓库 README 明确声明素材源自原作，仅限非商用学习分享。
- `PixelSRPG-Forge`：**不建议直接使用**。虽然仓库结构采用 MIT，但 README 明确写明“单个素材版权状态不确定，商业用途风险自负”。

因此，两者都不应直接进入本项目发布包。

## 当前项目策略

- 已下载并完成评估，仅作风格参考。
- 发布流程新增 `license_guard`，防止高风险来源素材被误打包：
  - `scripts/license_guard.py`
  - `scripts/release_check.py`
  - `scripts/build_windows.ps1`

## 推荐可商用替代方案

1. 继续使用已落地且明确授权素材（当前项目内的 CC0 包）。
2. 仅将上述仓库用于“配色、构图、UI层级、像素密度”参考，不直接拷贝图块。
3. 新增素材必须满足：
   - 授权明确（CC0 / CC-BY / 可商用条款清晰）
   - 保留本地 license 文件
   - 更新 `THIRD_PARTY_LICENSES.md`
